# UHS Course Catalogue — Complete Course Codes

Every course has a unique code in the format `{AREA}-{GRADE}[-SUFFIX]`.

- Student courses: `MATH-G1`
- Teachers Guides: `MATH-G1-TG`
- Revision books: `MATH-G1-REV`
- Diploma: `MATH-DIP`, `MATH-DIP-TG`, `MATH-DIP-REV`
- Exam prep: `MATH-G6-KPSEA`, `ENG-G9-KJSEA`, `PHY-G12-KCSE`

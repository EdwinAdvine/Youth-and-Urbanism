## Image Guidelines

- Format: PNG preferred, JPG for photos
- Resolution: 1920x1080 minimum for full-page screenshots
- Naming: Use kebab-case (lowercase with hyphens)
- Ensure no sensitive data (emails, passwords) is visible
- Use demo/seed data accounts for screenshots
- Capture both light and dark mode where relevant
